## –í –∞–Ω–∞–ª–∏—Ç–∏—á–µ—Å–∫–æ–º –ø–æ–¥—Ö–æ–¥–µ –º—ã —Ä–µ—à–∞–µ–º —É—Ä–∞–≤–Ω–µ–Ω–∏–µ 

$$\mathbf{\hat{\beta}} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{y}$$

Where:
- $\mathbf{\hat{\beta}}$ = vector of estimated coefficients $[\beta_0, \beta_1, ..., \beta_n]^T$
- $\mathbf{X}$ = design matrix (n √ó (p+1)) with intercept column
- $\mathbf{y}$ = vector of target values (n √ó 1)
- $\mathbf{X}^T$ = transpose of X
- $(\mathbf{X}^T\mathbf{X})^{-1}$ = inverse of $\mathbf{X}^T\mathbf{X}$

### Advantages of Analytical Solution

‚úÖ **Exact solution** - No approximation or iteration needed  
‚úÖ **Fast for small datasets** - One-step calculation  
‚úÖ **No hyperparameters** - No learning rate to tune  
‚úÖ **Guaranteed convergence** - Always finds the optimal solution  

### Disadvantages

‚ùå **Computational cost** - Expensive for large feature sets (matrix inversion is $O(p^3)$)  
‚ùå **Memory intensive** - Requires storing $\mathbf{X}^T\mathbf{X}$ matrix  
‚ùå **Numerical stability** - Can fail if $\mathbf{X}^T\mathbf{X}$ is singular or near-singular  
‚ùå **Not scalable** - Poor performance with millions of features  

### When to Use

- **Use Analytical Solution when:**
  - Number of features $p < 10,000$
  - Need exact solution
  - Dataset fits in memory
  - $\mathbf{X}^T\mathbf{X}$ is well-conditioned

## –í—Ç–æ—Ä–æ–π –º–µ—Ç–æ–¥ —ç—Ç–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –≥—Ä–∞–¥–∏–µ–Ω—Ç–Ω–æ–≥–æ —Å–ø—É—Å–∫–∞ 

–ü—Ä–∏ —ç—Ç–æ–º –º–µ—Ç–æ–¥–µ –º—ã —Ä–µ—à–∞–µ–º –∑–∞–¥–∞—á—É –º–∏–Ω–∏–º–∏–∑–∞—Ü–∏–∏ –Ω–µ–∫–æ—Ç–æ—Ä–æ–π —Ñ—É–Ω–∫—Ü–∏–∏ –ø–æ—Ç–µ—Ä—å. –¢–æ–µ—Å—Ç—å –º—ã —Å–ª—É—á–∞–π–Ω–æ –∏–Ω–∏—Ü–∏–∞–ª–∏–∑–∏—Ä—É–µ–º –≤–µ—Å–∞, –¥–µ–ª–∞–µ–º –ø—Ä–æ–≥–Ω–æ–∑, —Å—á–∏—Ç–∞–µ–º —Ñ—É–Ω–∫—Ü–∏—é –æ—à–∏–±–∫–∏, –±–µ—Ä–µ–º –≥—Ä–∞–¥–∏–µ–Ω—Ç –æ—Ç —Ñ—É–Ω–∫—Ü–∏–∏ –ø–æ—Ç–µ—Ä—å –∏ –¥–µ–ª–∞–µ–º —à–∞–≥ –≤ —Å—Ç–æ—Ä–æ–Ω—É –µ–≥–æ —É–º–µ–Ω—å—à–µ–Ω–∏—è. 

Initialize Œ≤ randomly or to zeros Set learning rate Œ± (e.g., 0.01) Set number of iterations (e.g., 1000)

For each iteration: 
1. Compute predictions: ≈∑ = XŒ≤ 
2. Compute error: error = ≈∑ - y

### Gradient Descent Update Rules

$$\beta_0 := \beta_0 - \alpha\frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i - y_i)$$

$$\beta_1 := \beta_1 - \alpha\frac{1}{n}\sum_{i=1}^{n}(\hat{y}_i - y_i)x_i$$

Where $\alpha$ is the learning rate.

–°–∞–º–æ–π –±–∞–∑–æ–≤–æ–π —Ñ—É–Ω–∫—Ü–∏–µ–π —Å—á–∏–∞—Ç–µ—Ç—Å—è MSE (Mean Square Error):

### Standard Form
$$J(\beta) = \frac{1}{2m}\sum_{i=1}^{m}(h_{\beta}(x^{(i)}) - y^{(i)})^2$$

Or equivalently:
$$J(\beta_0, \beta_1, ..., \beta_n) = \frac{1}{2m}\sum_{i=1}^{m}(\hat{y}^{(i)} - y^{(i)})^2$$

Where:
- $J(\beta)$ = cost function
- $m$ = number of training examples
- $h_{\beta}(x^{(i)})$ or $\hat{y}^{(i)}$ = predicted value for example $i$
- $y^{(i)}$ = actual value for example $i$
- $\frac{1}{2}$ = constant for mathematical convenience (cancels during derivative)

# –ß—Ç–æ —Ç–∞–∫–æ–µ —Ä–µ–≥—É–ª—è—Ä–∏–∑–∞—Ü–∏—è?

–†–µ–≥—É–ª—è—Ä–∏–∑–∞—Ü–∏—è –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –ø–µ—Ä–µ–¥–æ—Ç–≤—Ä–∞—â–µ–Ω–∏—è –ø–µ—Ä–µ–æ–±—É—á–µ–Ω–∏—è –≤ –º–æ–¥–µ–ª—è—Ö –±–∞–∑–æ–≤—ã–µ —Ä–µ–≥—É–ª—è—Ä–∏–∑–∞—Ü–∏–∏ —ç—Ç–æ L1 –∏ L2: 

# L2 Regularization (Ridge Regression)

### Properties
- **Shrinks** coefficients towards zero (but not exactly zero)
- **Reduces model complexity** by penalizing large coefficients
- **Handles multicollinearity** by making $\mathbf{X}^T\mathbf{X} + \lambda\mathbf{I}$ invertible
- **Differentiable** everywhere (smooth optimization)
- **Feature selection:** Does NOT perform feature selection (keeps all features)


### Cost Function
$$J(\beta) = \frac{1}{2m}\sum_{i=1}^{m}(h_{\beta}(x^{(i)}) - y^{(i)})^2 + \frac{\lambda}{2m}\sum_{j=1}^{n}\beta_j^2$$

Or in compact notation:
$$J(\beta) = MSE + \frac{\lambda}{2m}||\beta||_2^2$$

Where:
- $\lambda$ = regularization parameter (hyperparameter, $\lambda \geq 0$)
- $||\beta||_2^2 = \sum_{j=1}^{n}\beta_j^2$ = squared L2 norm
- Note: $\beta_0$ (intercept) is typically **not** regularized

### Matrix Form
$$J(\beta) = \frac{1}{2m}(\mathbf{X\beta} - \mathbf{y})^T(\mathbf{X\beta} - \mathbf{y}) + \frac{\lambda}{2m}\mathbf{\beta}^T\mathbf{\beta}$$

### Gradient for Gradient Descent
$$\frac{\partial J(\beta)}{\partial \beta_j} = \frac{1}{m}\sum_{i=1}^{m}(h_{\beta}(x^{(i)}) - y^{(i)}) \cdot x_j^{(i)} + \frac{\lambda}{m}\beta_j$$

### Vectorized Gradient
$$\nabla J(\beta) = \frac{1}{m}\mathbf{X}^T(\mathbf{X\beta} - \mathbf{y}) + \frac{\lambda}{m}\mathbf{\beta}$$

### Update Rule (Gradient Descent)
$$\beta_j := \beta_j - \alpha\left[\frac{1}{m}\sum_{i=1}^{m}(h_{\beta}(x^{(i)}) - y^{(i)}) \cdot x_j^{(i)} + \frac{\lambda}{m}\beta_j\right]$$

Which can be rewritten as:
$$\beta_j := \beta_j\left(1 - \alpha\frac{\lambda}{m}\right) - \alpha\frac{1}{m}\sum_{i=1}^{m}(h_{\beta}(x^{(i)}) - y^{(i)}) \cdot x_j^{(i)}$$

### Analytical Solution (Normal Equation)
$$\mathbf{\hat{\beta}} = (\mathbf{X}^T\mathbf{X} + \lambda\mathbf{I})^{-1}\mathbf{X}^T\mathbf{y}$$

Where $\mathbf{I}$ is the identity matrix.

# L1 Regularization (Lasso Regression)

## Properties
- **Shrinks** some coefficients **exactly to zero** (sparse solutions)
- **Performs automatic feature selection**
- **Not differentiable** at zero (requires subgradient methods)
- **No closed-form solution** (must use iterative methods)
- Useful when you have many features and want to identify the most important ones

## Cost Function
$$J(\beta) = \frac{1}{2m}\sum_{i=1}^{m}(h_{\beta}(x^{(i)}) - y^{(i)})^2 + \frac{\lambda}{m}\sum_{j=1}^{n}|\beta_j|$$

Or in compact notation:
$$J(\beta) = MSE + \frac{\lambda}{m}||\beta||_1$$

Where:
- $\lambda$ = regularization parameter (hyperparameter, $\lambda \geq 0$)
- $||\beta||_1 = \sum_{j=1}^{n}|\beta_j|$ = L1 norm (Manhattan distance)
- Note: $\beta_0$ (intercept) is typically **not** regularized

## Matrix Form
$$J(\beta) = \frac{1}{2m}(\mathbf{X\beta} - \mathbf{y})^T(\mathbf{X\beta} - \mathbf{y}) + \frac{\lambda}{m}\sum_{j=1}^{n}|\beta_j|$$

## Subgradient for Gradient Descent
$$\frac{\partial J(\beta)}{\partial \beta_j} = \frac{1}{m}\sum_{i=1}^{m}(h_{\beta}(x^{(i)}) - y^{(i)}) \cdot x_j^{(i)} + \frac{\lambda}{m} \cdot \text{sign}(\beta_j)$$

Where:
$$\text{sign}(\beta_j) = \begin{cases}
+1 & \text{if } \beta_j > 0 \\
0 & \text{if } \beta_j = 0 \\
-1 & \text{if } \beta_j < 0
\end{cases}$$

## Update Rule (Gradient Descent)

### –ö–æ–≥–¥–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å
‚úÖ –°–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –∫–ª–∞—Å—Å—ã  
‚ùå **–ù–ï –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å** –ø—Ä–∏ –Ω–µ—Å–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –∫–ª–∞—Å—Å–∞—Ö

**Precision** ($\frac{TP}{TP+FP}$) - –æ—Ü–µ–Ω–∏–≤–∞–µ—Ç –∫–∞—á–µ—Å—Ç–≤–æ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã—Ö –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏–π. –ü–æ–º–æ–≥–∞–µ—Ç –∏–∑–±–µ–≥–∞—Ç—å –æ—à–∏–±–æ–∫ —Ç–∏–ø–∞ False Positive.

### –ò–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏—è
- –ò–∑ –≤—Å–µ—Ö –ø—Ä–∏–º–µ—Ä–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –º–æ–¥–µ–ª—å –ø—Ä–µ–¥—Å–∫–∞–∑–∞–ª–∞ –∫–∞–∫ **–ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–µ**, –∫–∞–∫–∞—è –¥–æ–ª—è –¥–µ–π—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–∞?
- –û—Ç–≤–µ—á–∞–µ—Ç –Ω–∞ –≤–æ–ø—Ä–æ—Å: **"–ù–∞—Å–∫–æ–ª—å–∫–æ –º–æ–∂–Ω–æ –¥–æ–≤–µ—Ä—è—Ç—å –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–º—É –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏—é?"**

### –ö–æ–≥–¥–∞ –≤–∞–∂–Ω–∞
- –ö–æ–≥–¥–∞ **–ª–æ–∂–Ω—ã–µ —Å—Ä–∞–±–∞—Ç—ã–≤–∞–Ω–∏—è (FP) –¥–æ—Ä–æ–≥–æ –æ–±—Ö–æ–¥—è—Ç—Å—è**
- –°–ø–∞–º-—Ñ–∏–ª—å—Ç—Ä (–Ω–µ —Ö–æ—Ç–∏–º –ø–æ–º–µ—á–∞—Ç—å –≤–∞–∂–Ω—ã–µ –ø–∏—Å—å–º–∞ –∫–∞–∫ —Å–ø–∞–º)
- –ú–µ–¥–∏—Ü–∏–Ω—Å–∫–∞—è –¥–∏–∞–≥–Ω–æ—Å—Ç–∏–∫–∞ (–Ω–µ —Ö–æ—Ç–∏–º –ª–æ–∂–Ω–æ –¥–∏–∞–≥–Ω–æ—Å—Ç–∏—Ä–æ–≤–∞—Ç—å –±–æ–ª–µ–∑–Ω—å)
- –†–µ–∫–æ–º–µ–Ω–¥–∞—Ç–µ–ª—å–Ω—ã–µ —Å–∏—Å—Ç–µ–º—ã (–Ω–µ —Ö–æ—Ç–∏–º —Ä–µ–∫–æ–º–µ–Ω–¥–æ–≤–∞—Ç—å –Ω–µ—Ä–µ–ª–µ–≤–∞–Ω—Ç–Ω–æ–µ)

**Recall** ($\frac{TP}{TP+FN}$) - –æ—Ü–µ–Ω–∏–≤–∞–µ—Ç –¥–æ–ª—é –≤–µ—Ä–Ω–æ –Ω–∞–π–¥–µ–Ω–Ω—ã—Ö –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã—Ö –æ—Ç–≤–µ—Ç–æ–≤ –∏–∑ –≤—Å–µ—Ö –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã—Ö –æ—Ç–≤–µ—Ç–æ–≤. –ü–æ–º–æ–≥–∞–µ—Ç –∏–∑–±–µ–≥–∞—Ç—å –æ—à–∏–±–æ–∫ —Ç–∏–ø–∞ False Negative.

### –ò–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏—è
- –ò–∑ –≤—Å–µ—Ö **–¥–µ–π—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã—Ö** –ø—Ä–∏–º–µ—Ä–æ–≤, –∫–∞–∫—É—é –¥–æ–ª—é –º–æ–¥–µ–ª—å —Å–º–æ–≥–ª–∞ –Ω–∞–π—Ç–∏?
- –û—Ç–≤–µ—á–∞–µ—Ç –Ω–∞ –≤–æ–ø—Ä–æ—Å: **"–ù–∞—Å–∫–æ–ª—å–∫–æ —Ö–æ—Ä–æ—à–æ –º–æ–¥–µ–ª—å –Ω–∞—Ö–æ–¥–∏—Ç –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–µ –ø—Ä–∏–º–µ—Ä—ã?"**

### –ö–æ–≥–¥–∞ –≤–∞–∂–Ω–∞
- –ö–æ–≥–¥–∞ **–ø—Ä–æ–ø—É—Å–∫–∏ (FN) –¥–æ—Ä–æ–≥–æ –æ–±—Ö–æ–¥—è—Ç—Å—è**
- –ú–µ–¥–∏—Ü–∏–Ω—Å–∫–∞—è –¥–∏–∞–≥–Ω–æ—Å—Ç–∏–∫–∞ —Å–µ—Ä—å—ë–∑–Ω—ã—Ö –∑–∞–±–æ–ª–µ–≤–∞–Ω–∏–π (–Ω–µ —Ö–æ—Ç–∏–º –ø—Ä–æ–ø—É—Å—Ç–∏—Ç—å –±–æ–ª—å–Ω–æ–≥–æ)
- –û–±–Ω–∞—Ä—É–∂–µ–Ω–∏–µ –º–æ—à–µ–Ω–Ω–∏—á–µ—Å—Ç–≤–∞ (–ª—É—á—à–µ –ø—Ä–æ–≤–µ—Ä–∏—Ç—å –ª–∏—à–Ω–µ–µ, —á–µ–º –ø—Ä–æ–ø—É—Å—Ç–∏—Ç—å –º–æ—à–µ–Ω–Ω–∏–∫–∞)
- –ü–æ–∏—Å–∫ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏ (—Ö–æ—Ç–∏–º –Ω–∞–π—Ç–∏ –≤—Å–µ —Ä–µ–ª–µ–≤–∞–Ω—Ç–Ω—ã–µ –¥–æ–∫—É–º–µ–Ω—Ç—ã)


**F1 Score** ($\frac{2TP}{2TP+FP+FN}$) - –≥–∞—Ä–º–æ–Ω–∏—á–µ—Å–∫–æ–µ —Å—Ä–µ–¥–Ω–µ–µ –º–µ–∂–¥—É Precision –∏ Recall –ø–æ–º–æ–≥–∞–µ—Ç –ø–æ–Ω—è—Ç—å –±–∞–ª–∞–Ω—Å –∫–ª–∞—Å—Å–æ–≤. 

  - –î–∏–∞–ø–∞–∑–æ–Ω: $[0, 1]$ –∏–ª–∏ $[0\%, 100\%]$
  - –õ—É—á—à–µ ‚Üí 1 (100%)
  - **–®—Ç—Ä–∞—Ñ—É–µ—Ç** –º–æ–¥–µ–ª–∏ —Å –±–æ–ª—å—à–∏–º –¥–∏—Å–±–∞–ª–∞–Ω—Å–æ–º –º–µ–∂–¥—É Precision –∏ Recall

### –ö–æ–≥–¥–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å
- –ù—É–∂–µ–Ω –±–∞–ª–∞–Ω—Å –º–µ–∂–¥—É Precision –∏ Recall
- –ù–µ—Å–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –∫–ª–∞—Å—Å—ã
- –ö–æ–≥–¥–∞ –∏ FP, –∏ FN –æ–¥–∏–Ω–∞–∫–æ–≤–æ –≤–∞–∂–Ω—ã

**Specificity** ($\frac{TN}{TN+FP}$ ) -  –æ—Ü–µ–Ω–∏–≤–∞–µ—Ç –¥–æ–ª—é –≤–µ—Ä–Ω–æ –Ω–∞–π–¥–µ–Ω–Ω—ã—Ö –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã—Ö –æ—Ç–≤–µ—Ç–æ–≤ –∏–∑ –≤—Å–µ—Ö –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã—Ö –æ—Ç–≤–µ—Ç–æ–≤.

# –ß—Ç–æ —Ç–∞–∫–æ–µ ROC-AUC –∏ PR-AUC? –ö–∞–∫–∏–µ –ø–ª—é—Å—ã –∏ –º–∏–Ω—É—Å—ã?

**ROC-AUC (Receiver Operating Characteristic - Area Under Curve)** - –¥–ª—è –æ—Ü–µ–Ω–∫–∏ –ø—Ä–∏ —Ä–∞–∑–Ω—ã—Ö –ø–æ—Ä–æ–≥–∞—Ö. –ü–æ–∑–≤–æ–ª—è–µ—Ç –≤ –æ—Ü–µ–Ω–∏—Ç—å –æ–±—É—á–∞—é—â—É—é —Å–ø–æ—Å–æ–±–Ω–æ—Å—Ç—å –º–æ–¥–µ–ª–∏ –æ—Ç—Ä–∞–∂–∞—è –Ω–∞—Å–∫–æ–ª—å–∫–æ —Ö–æ—Ä–æ—à–æ –º–æ–¥–µ–ª—å –æ—Ç—Ä–∞–Ω–∂–∏—Ä—É–µ—Ç –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–µ –∏ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–µ –∫–ª–∞—Å—Å—ã.   

- **ROC-–∫—Ä–∏–≤–∞—è:** –≥—Ä–∞—Ñ–∏–∫ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ TPR (Recall) –æ—Ç FPR (Specificity) –ø—Ä–∏ —Ä–∞–∑–ª–∏—á–Ω—ã—Ö –ø–æ—Ä–æ–≥–∞—Ö –∫–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ü–∏–∏
- **AUC:** –ø–ª–æ—â–∞–¥—å –ø–æ–¥ ROC-–∫—Ä–∏–≤–æ–π (0 –¥–æ 1) - –º–æ–∂–Ω–æ –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∏—Ä–æ–≤–∞—Ç—å –∫–∞–∫ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ç–æ–≥–æ –ø—Ä–∞–≤–∏–ª—å–Ω–æ –ª–∏ –º–æ–¥–µ–ª—å –æ—Ç—Ä–∞–Ω–∂–∏—Ä—É–µ—Ç –¥–≤–∞ —Ä—è–¥–∞ –∫–ª–∞—Å—Å–æ–≤. 

$$TPR = \frac{TP}{TP + FN} = Recall$$

$$FPR = \frac{FP}{FP + TN} = 1 - Specificity$$

### –ò–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏—è AUC
- **1.0** ‚Äî –∏–¥–µ–∞–ª—å–Ω–∞—è –º–æ–¥–µ–ª—å
- **0.5** ‚Äî —Å–ª—É—á–∞–π–Ω–æ–µ —É–≥–∞–¥—ã–≤–∞–Ω–∏–µ

### ‚úÖ –ü–ª—é—Å—ã
- **–ù–µ–∑–∞–≤–∏—Å–∏–º–æ—Å—Ç—å –æ—Ç threshold** ‚Äî –æ—Ü–µ–Ω–∏–≤–∞–µ—Ç –º–æ–¥–µ–ª—å –ø—Ä–∏ –≤—Å–µ—Ö –ø–æ—Ä–æ–≥–∞—Ö
- **–ò–Ω–≤–∞—Ä–∏–∞–Ω—Ç–Ω–æ—Å—Ç—å –∫ –¥–∏—Å–±–∞–ª–∞–Ω—Å—É –∫–ª–∞—Å—Å–æ–≤** ‚Äî –Ω–µ –∑–∞–≤–∏—Å–∏—Ç –æ—Ç —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è –∫–ª–∞—Å—Å–æ–≤
- **–°—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–∞—è –º–µ—Ç—Ä–∏–∫–∞** ‚Äî —à–∏—Ä–æ–∫–æ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –∏ –ø–æ–Ω—è—Ç–Ω–∞
- **–•–æ—Ä–æ—à–∞ –¥–ª—è —Å–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö**
- **–õ–µ–≥–∫–æ –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∏—Ä–æ–≤–∞—Ç—å:** –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å, —á—Ç–æ –º–æ–¥–µ–ª—å –ø—Ä–∞–≤–∏–ª—å–Ω–æ —Ä–∞–Ω–∂–∏—Ä—É–µ—Ç —Å–ª—É—á–∞–π–Ω—ã–µ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–π –∏ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–π –ø—Ä–∏–º–µ—Ä—ã

### ‚ùå –ú–∏–Ω—É—Å—ã
- **–ü–µ—Ä–µ–æ—Ü–µ–Ω–∫–∞ –ø—Ä–∏ –Ω–µ—Å–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –∫–ª–∞—Å—Å–∞—Ö** ‚Äî –º–æ–∂–µ—Ç –ø–æ–∫–∞–∑—ã–≤–∞—Ç—å —Ö–æ—Ä–æ—à–∏–µ —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ã –¥–∞–∂–µ –ø—Ä–∏ –ø–ª–æ—Ö–æ–º –∫–∞—á–µ—Å—Ç–≤–µ –Ω–∞ —Ä–µ–¥–∫–æ–º –∫–ª–∞—Å—Å–µ
- **–ù–µ –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç –∞–±—Å–æ–ª—é—Ç–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è** ‚Äî —Ç–æ–ª—å–∫–æ –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ–µ –∫–∞—á–µ—Å—Ç–≤–æ
- **–û–ø—Ç–∏–º–∏—Å—Ç–∏—á–Ω–∞ –¥–ª—è –Ω–µ—Å–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö** ‚Äî FPR –º–æ–∂–µ—Ç –±—ã—Ç—å –Ω–∏–∑–∫–∏–º –ø—Ä–æ—Å—Ç–æ –∏–∑-–∑–∞ –±–æ–ª—å—à–æ–≥–æ TN
- **–ù–µ —É—á–∏—Ç—ã–≤–∞–µ—Ç —Å—Ç–æ–∏–º–æ—Å—Ç—å –æ—à–∏–±–æ–∫** ‚Äî FP –∏ FN —Ä–∞–≤–Ω–æ–∑–Ω–∞—á–Ω—ã

**PR-AUC (Precision-Recall - Area Under Curve)** - –¥–ª—è —Å–∏–ª—å–Ω–æ –Ω–µ—Å–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö

- **PR-–∫—Ä–∏–≤–∞—è:** –≥—Ä–∞—Ñ–∏–∫ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ Precision –æ—Ç Recall –ø—Ä–∏ —Ä–∞–∑–ª–∏—á–Ω—ã—Ö –ø–æ—Ä–æ–≥–∞—Ö
- **AUC:** –ø–ª–æ—â–∞–¥—å –ø–æ–¥ PR-–∫—Ä–∏–≤–æ–π (0 –¥–æ 1)

### –ò–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏—è AUC
- **1.0** ‚Äî –∏–¥–µ–∞–ª—å–Ω–∞—è –º–æ–¥–µ–ª—å
- **> 0.5** ‚Äî –æ–±—ã—á–Ω–æ —Ö–æ—Ä–æ—à–æ

### ‚úÖ –ü–ª—é—Å—ã
- **–§–æ–∫—É—Å –Ω–∞ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–º –∫–ª–∞—Å—Å–µ** ‚Äî –∏–≥–Ω–æ—Ä–∏—Ä—É–µ—Ç TN
- **–õ—É—á—à–µ –¥–ª—è –Ω–µ—Å–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö** ‚Äî —á—É–≤—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–∞ –∫ –∫–∞—á–µ—Å—Ç–≤—É –Ω–∞ —Ä–µ–¥–∫–æ–º –∫–ª–∞—Å—Å–µ
- **–ü–æ–∫–∞–∑—ã–≤–∞–µ—Ç —Ä–µ–∞–ª—å–Ω—É—é –ø—Ä–æ–∏–∑–≤–æ–¥–∏—Ç–µ–ª—å–Ω–æ—Å—Ç—å** ‚Äî –Ω–µ "–ø—Ä—è—á–µ—Ç" –ø—Ä–æ–±–ª–µ–º—ã –∑–∞ –±–æ–ª—å—à–∏–º –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ–º TN
- **–ë–æ–ª–µ–µ –∫–æ–Ω—Å–µ—Ä–≤–∞—Ç–∏–≤–Ω–∞—è –æ—Ü–µ–Ω–∫–∞**
- **–†–µ–ª–µ–≤–∞–Ω—Ç–Ω–∞ –¥–ª—è –∑–∞–¥–∞—á —Å —Ä–µ–¥–∫–∏–º –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–º –∫–ª–∞—Å—Å–æ–º** (fraud detection, –º–µ–¥–∏—Ü–∏–Ω—Å–∫–∞—è –¥–∏–∞–≥–Ω–æ—Å—Ç–∏–∫–∞)

### ‚ùå –ú–∏–Ω—É—Å—ã
- **–°–ª–æ–∂–Ω–µ–µ –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∏—Ä–æ–≤–∞—Ç—å** ‚Äî –Ω–µ—Ç –∏–Ω—Ç—É–∏—Ç–∏–≤–Ω–æ–π –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–Ω–æ–π –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏–∏
- **–ó–∞–≤–∏—Å–∏—Ç –æ—Ç —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è –∫–ª–∞—Å—Å–æ–≤** ‚Äî –Ω–µ–ª—å–∑—è —Å—Ä–∞–≤–Ω–∏–≤–∞—Ç—å –º–µ–∂–¥—É —Ä–∞–∑–Ω—ã–º–∏ –¥–∞—Ç–∞—Å–µ—Ç–∞–º–∏

# –ö–∞–∫–∏–µ –º–µ—Ç–æ–¥—ã —Å—É—â–µ—Å—Ç–≤—É—é—Ç –¥–ª—è –º–Ω–æ–≥–æ–∫–ª–∞—Å—Å–æ–≤–æ–π –∫–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ü–∏–∏?

–¢—É—Ç —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –¥–≤–∞ –æ—Å–Ω–æ–≤–Ω—ã—Ö –ø–æ–¥—Ö–æ–¥–∞: 

## One-vs-Rest (OvR) / One-vs-All (OvA)

**–ü—Ä–∏–Ω—Ü–∏–ø:**
- –û–±—É—á–∞–µ—Ç—Å—è K –±–∏–Ω–∞—Ä–Ω—ã—Ö –∫–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ç–æ—Ä–æ–≤ (K = –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∫–ª–∞—Å—Å–æ–≤)
- –ö–∞–∂–¥—ã–π –∫–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ç–æ—Ä: "–∫–ª–∞—Å—Å i" vs "–≤—Å–µ –æ—Å—Ç–∞–ª—å–Ω—ã–µ"
- –ü—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏–µ: –∫–ª–∞—Å—Å —Å –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–π –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å—é

**–§–æ—Ä–º—É–ª–∞:**
$$\hat{y} = \arg\max_{k \in \{1,...,K\}} f_k(x)$$

**–ü–ª—é—Å—ã:**
- ‚úÖ –ü—Ä–æ—Å—Ç–æ—Ç–∞
- ‚úÖ –≠—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ—Å—Ç—å (K –º–æ–¥–µ–ª–µ–π)
- ‚úÖ –ë—ã—Å—Ç—Ä–æ–µ –æ–±—É—á–µ–Ω–∏–µ

**–ú–∏–Ω—É—Å—ã:**
- ‚ùå –î–∏—Å–±–∞–ª–∞–Ω—Å –∫–ª–∞—Å—Å–æ–≤ (1 vs –≤—Å–µ)
- ‚ùå –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ –Ω–µ –æ—Ç–∫–∞–ª–∏–±—Ä–æ–≤–∞–Ω—ã


## One-vs-One (OvO) / All-vs-All

**–ü—Ä–∏–Ω—Ü–∏–ø:**
- –û–±—É—á–∞–µ—Ç—Å—è $\frac{K(K-1)}{2}$ –±–∏–Ω–∞—Ä–Ω—ã—Ö –∫–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ç–æ—Ä–æ–≤
